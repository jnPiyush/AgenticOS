## 1. Overview

PawConnect is a social networking platform for dog owners to connect, schedule playdates, find services, and attend events. The design focuses on a "dog-first" approach, where profiles represent dogs and their personalities.

**Core Experience Pillars:**
- **Community**: Fostering local connections.
- **Trust**: Verified profiles and reviews.
- **Joy**: Playful, engaging interface.
- **Ease**: Simple discovery and booking.

## 2. User Research

### Personas
*Refer to PRD Section 2 for full details.*
1.  **Sarah (Active Owner)**: Mobile-heavy, wants quick matching and active social feed.
2.  **Mike (Service Provider)**: Desktop/Tablet user, manages bookings and business profile.
3.  **Linda (Organizer)**: Needs clear event management tools.

## 3. User Flows

### 3.1 Playdate Matching Flow
1.  **Nav**: Tap "Playdates" tab.
2.  **Browse**: View card stack of compatible dogs (Match % displayed).
3.  **Action**: Tap "Heart" (Request) or "X" (Skip).
4.  **Request**: If Heart -> Open Plan Playdate Modal (Time/Location).
5.  **Confirm**: Send Request -> Success Toast.

### 3.2 Service Booking Flow
1.  **Nav**: Tap "Services" tab.
2.  **Filter**: Select "Grooming" + "Near Me".
3.  **Select**: Tap Provider Card -> View Profile.
4.  **Book**: Tap "Book Now" -> Select Service -> Select Time.
5.  **Confirm**: Review & Submit -> Confirmation Screen.

## 5. Design System

### Colors
- **Primary**: `#FF6B6B` (Playful Red)
- **Secondary**: `#4ECDC4` (Fresh Teal)
- **Neutral**: `#F7FFF7` (Off-white bg), `#292F36` (Dark Text)
- **Accent**: `#FFE66D` (Sunny Yellow)

### Typography
- **Headings**: 'Nunito', Rounded sans-serif (Friendly)
- **Body**: 'Open Sans' (Readable)

## 7. Accessibility (WCAG 2.1 AA)

- **Color Contrast**: All text meets 4.5:1 ratio against background.
- **Focus States**: Visible focus indicators for keyboard navigation.
- **Alt Text**: All user images require alt text input.
- **Semantic HTML**: Proper use of `<nav>`, `<main>`, `<article>`, `<button>`.

## 8. Responsive Design

- **Mobile First**: Built for touch targets (min 44px).
- **Desktop**: Expands to grid layout for Feed/Services.
